## ✅ Prompt: Create a Unique, High-End Minimalist Website with Scroll-Based Motion

**Prompt:**

Create a modern, high-quality, minimalist website with a unique and creative design concept. The design must feel premium, clean, and visually refined with strong typography and intentional white space.

### 🎨 Design Direction
- Style: Minimalist, elegant, futuristic
- Color palette: Neutral base (white, off-white, or soft beige) with one bold accent color
- Typography: Large bold headline font paired with a clean sans-serif body font
- Layout: Grid-based, asymmetric but balanced
- Strong visual hierarchy
- Generous spacing and breathing room

---

### ✨ Scroll & Interaction Effects
Add smooth, tasteful motion effects that enhance the experience without overwhelming it:

- Fade + upward reveal animations on scroll
- Parallax scrolling sections
- Text reveal animation (split text or masked reveal)
- Image scale-in on scroll
- Horizontal scrolling section within vertical scroll
- Sticky section with dynamic content change
- Subtle hover animations (buttons, links, images)
- Smooth scrolling enabled
- Micro-interactions on buttons

All animations must feel smooth, modern, and premium — not flashy.

---

### 🧠 Unique Concept Idea
Incorporate a creative storytelling approach:

Option examples:
- “Journey-based scroll” (each scroll feels like a chapter)
- Split-screen layout with scrolling contrast
- Interactive typography hero section
- Floating minimal 3D shapes with subtle motion
- Cursor-based interaction (light magnetic effect)

Make the website feel different from generic templates.

---

### 📄 Website Structure
1. **Hero Section**
   - Large bold headline
   - Minimal subtext
   - Clean CTA button
   - Subtle animated background element

2. **About / Story Section**
   - Split layout (text + image or visual)
   - Scroll reveal animation

3. **Services / Features**
   - Grid layout
   - Hover animation per card
   - Icon or minimal visual element

4. **Showcase / Portfolio**
   - Masonry or grid layout
   - Smooth hover preview
   - Optional horizontal scroll section

5. **Testimonial / Social Proof**
   - Minimal card design
   - Soft animation

6. **Final CTA Section**
   - Bold statement
   - Clean button
   - Strong visual contrast

7. **Minimal Footer**
   - Simple navigation
   - Social links
   - Copyright

---

### ⚙️ Technical Requirements
- Fully responsive (mobile-first)
- Fast loading & optimized performance
- SEO-friendly structure
- Accessibility compliant
- Smooth scrolling enabled
- Lightweight animations (no heavy lag)

If generating code:
- Use modern HTML5 + CSS3 + JavaScript
- Or React / Next.js with Tailwind CSS
- Use GSAP or Framer Motion for animations
- Clean, well-structured code

---

### 🎯 Overall Feel
The final website should feel:
- Premium
- Calm
- Modern
- Innovative
- Thoughtfully designed
- Not template-like
- Visually impressive but simple
